## Installation “propre” de Windows 11 sans logiciels préinstallés

Lors d’une installation *propre* de Windows 11 — c'est-à-dire en téléchargeant l’ISO (image du DVD d’installation) directement depuis les serveurs de Microsoft, puis en l’installant via une clé USB — il est possible d’indiquer une localisation non reconnue par les serveurs de Microsoft. Cela évite l’installation automatique de logiciels liés à votre localisation réelle.

---

### Astuce : sélectionner *Anglais (Européen)*

En sélectionnant **"Anglais (Européen)"**, Windows n'installera pas de logiciels tiers.

> © Les Numériques

Pour ce faire, lors du premier écran de choix de langue, indiquez dans le champ **Format horaire et monétaire** :  
**“Anglais (Europe)”**.

Le processus d’installation affichera ensuite une erreur qu’il suffira d’**ignorer**. L’installation poursuivra alors normalement.

Ensuite, choisissez **l’édition Professionnelle**.

---

### Résultat

Arrivé sur le bureau de Windows 11, vous profiterez d’un menu Démarrer **dépourvu de toute application préinstallée**, qu’il s’agisse de services de SVOD, de messageries ou même de la suite Office.

---

### Comparatif
- **Installation en Anglais Européen** : aucune application préinstallée.  
- **Installation traditionnelle** : installation de logiciels tiers selon la localisation.

---

### Finalisation

Pour terminer l’installation, remettez simplement la localisation sur **France** dans les paramètres (si Windows ne l’a pas fait automatiquement) et installez ensuite uniquement les logiciels dont vous avez besoin depuis le **Microsoft Store**.
